## Project Overview

With the rise of generative AI tools, identifying whether text is written by humans or generated by AI has become increasingly important.  
This project builds a machine learning-based classifier that analyzes textual patterns and predicts the origin of the text with a confidence score.

---

## How It Works

1. Input text is cleaned (lowercasing, stopword removal, lemmatization)
2. Cleaned text is transformed using TF-IDF Vectorizer
3. Logistic Regression model predicts:
   - AI Generated
   - Human Written
4. Confidence score is calculated using prediction probabilities
5. Result is displayed in a Streamlit web interface
